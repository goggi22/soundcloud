# SoundCloud OAuth Redirect Page

Статический сайт для GitHub Pages, который принимает код из redirect URI и копирует его одним нажатием.

## Как работает

1. SoundCloud редиректит на страницу вида `https://your-site.example/?code=...`
2. Страница отображает код и предлагает скопировать его в буфер обмена.
3. Пользователь вставляет код в бот командой `.sccode <code>`.

## Развертывание

Просто разместите содержимое этой папки на GitHub Pages или другом static-хостинге.
